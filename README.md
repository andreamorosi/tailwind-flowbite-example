# ottima-flowbite

A Vite + Tailwind CSS v4 + Flowbite project, built entirely through prompts with **Claude Sonnet 4.6** via Claude Code.

## Stack

- **Vite** — build tool / dev server
- **Tailwind CSS v4** — utility-first CSS (`@tailwindcss/postcss`)
- **Flowbite** — component library on top of Tailwind

## How it was built

Prompt sequence used to generate this project:

1. Tailwind v4 installation
2. Install Flowbite and wire it up (theme, plugin, JS)
3. Add a Flowbite hero section to verify the integration
4. Replicate the [buona.it](https://www.buona.it/) homepage with Flowbite + Tailwind (videos replaced with images, real images replaced with placeholders)
